## 1. การเริ่มต้นใช้งาน

### 1.1 เข้าสู่ระบบ (Login)

1. เปิดเว็บเบราว์เซอร์ แล้วไปที่ URL ของระบบ
2. กรอก **อีเมล** และ **รหัสผ่าน**
3. กดปุ่ม **"เข้าสู่ระบบ"**
4. ระบบจะพาไปยังหน้า Dashboard โดยอัตโนมัติ

### 1.2 ลืมรหัสผ่าน

1. คลิก **"ลืมรหัสผ่าน"** ที่หน้า Login
2. กรอก **อีเมล** ที่ใช้ลงทะเบียน
3. กดปุ่ม **"ส่งลิงก์รีเซ็ต"**
4. เปิดอีเมล แล้วคลิกลิงก์เพื่อตั้งรหัสผ่านใหม่ (ลิงก์มีอายุ 1 ชั่วโมง)

### 1.3 ส่วนประกอบของหน้าจอ

| ส่วน | คำอธิบาย |
|------|----------|
| **เมนูด้านซ้าย (Sidebar)** | นำทางไปยังหน้าต่างๆ (Dashboard, นัดหมาย, ผลตรวจ ฯลฯ) |
| **แถบด้านบน (Header)** | ข้อมูลผู้ใช้, แจ้งเตือน, ออกจากระบบ |
| **ปุ่ม ? (มุมขวาล่าง)** | ดูคำแนะนำการใช้งานของหน้าปัจจุบัน |
| **ป้ายตัวเลขสีแดง** | แสดงจำนวนรายการใหม่ที่ยังไม่ได้ตรวจสอบ |

## 2. หน้า Dashboard

หน้าหลักแสดงภาพรวมของระบบ

### ฟีเจอร์หลัก:
- **ค้นหา**: พิมพ์ HN หรือชื่อผู้ป่วยเพื่อค้นหาอย่างรวดเร็ว
- **รายการล่าสุด**: แสดงผู้ป่วยที่มีการอัปเดตสถานะล่าสุด
- **ปุ่มลัด**: กดปุ่ม "ลงทะเบียนผู้ป่วยใหม่" เพื่อเพิ่มผู้ป่วยใหม่ทันที
- **สรุปจำนวน**: ดูจำนวนผู้ป่วยแยกตามสถานะ

## 3. การจัดการผู้ป่วย

### 3.1 ลงทะเบียนผู้ป่วยใหม่ (`/patients/add`)

1. กดปุ่ม **"ลงทะเบียนผู้ป่วยใหม่"** จาก Dashboard หรือเมนูด้านซ้าย
2. กรอกข้อมูล:
   - **HN** (Hospital Number) — จำเป็น
   - **ชื่อ-นามสกุล** — จำเป็น
   - **เพศ, อายุ, หมู่เลือด** — เสริม
   - **โรคประจำตัว, แพ้ยา** — เสริม
3. กดปุ่ม **"บันทึก"**
4. ผู้ป่วยจะปรากฏในรายการและปฏิทิน

### 3.2 ดูข้อมูลผู้ป่วย (`/patients/[hn]`)

- ดูข้อมูลทั่วไป: HN, ชื่อ, สถานะปัจจุบัน
- อัปเดตสถานะ: กดปุ่มเพื่อเปลี่ยนสถานะตามขั้นตอน
- ดูประวัตินัดหมายและผลตรวจ

### 3.3 ประวัติผู้ป่วย (`/history`)

- ค้นหาผู้ป่วยด้วย HN หรือชื่อ
- คลิกเพื่อดูรายละเอียด:
  - ผลตรวจเลือดย้อนหลัง พร้อมปุ่ม "ดูผล"
  - ตัวเตือน **8 วัน** หลังตรวจครั้งล่าสุด
  - ประวัตินัดหมาย
  - อัปเดตสถานะผู้ป่วย

### 3.4 สถานะผู้ป่วย (Workflow)

```
ลงทะเบียน → นัดหมาย → รอตรวจ → เจาะเลือด → กำลังจัดส่ง → กำลังตรวจ → เสร็จสิ้น → รายงานผล
```

| สถานะ | คำอธิบาย |
|-------|----------|
| ลงทะเบียน | เพิ่งสร้างในระบบ |
| นัดหมาย | มีวันนัดตรวจแล้ว |
| รอตรวจ | รอเข้าห้องตรวจ |
| เจาะเลือด | กำลังเจาะเลือด |
| กำลังจัดส่ง | ส่งตัวอย่างเลือดไป Lab |
| กำลังตรวจ | Lab กำลังตรวจวิเคราะห์ |
| เสร็จสิ้น | ผลตรวจพร้อมแล้ว |
| รายงานผล | พิมพ์ผลตรวจแล้ว |

## 4. ปฏิทินนัดหมาย (`/appointments`)

- **มุมมอง**: เปลี่ยนเป็น รายเดือน / รายสัปดาห์ / รายวัน
- **ดูรายละเอียด**: คลิกที่นัดหมายเพื่อดูข้อมูลผู้ป่วย
- **สีสถานะ**: แถบสีแสดงสถานะของผู้ป่วย
- **เพิ่มนัดหมาย**: คลิกที่วันที่ว่างเพื่อนัดหมายใหม่

## 5. ผลตรวจเลือด

### 5.1 หน้ารวมผลตรวจ (`/results`)

แสดงรายชื่อผู้ป่วยที่มีผลตรวจพร้อมแล้ว (สถานะ "เสร็จสิ้น" หรือ "รายงานผล")

- **ดูผล**: คลิกเพื่อดูรายละเอียดผลตรวจ
- **จุดสี**: แสดงสรุปผล (🟢 ปกติ / 🟡 ผิดปกติ / 🔴 วิกฤต)
- **พิมพ์แล้ว**: แสดงป้ายสีเขียวสำหรับรายที่พิมพ์ผลแล้ว
- **เช็ควันที่**: กรอกวันที่ (พ.ศ.) เพื่อคำนวณว่าตรวจมากี่วันแล้ว

### 5.2 ผลตรวจรายคน (`/results/[hn]`)

- **ดูไฟล์ผลตรวจ**: แสดงรูปภาพหรือ PDF ของผลตรวจ
- **สรุปผล**: 🟢 ปกติ / 🟡 ผิดปกติ / 🔴 วิกฤต
- **ประวัติ**: ดูผลตรวจย้อนหลัง (คลิกเพื่อสลับดู)
- **อนุญาตให้พิมพ์**: แพทย์/พยาบาลกดเพื่ออนุมัติก่อนพิมพ์
- **พิมพ์ผลตรวจ**: หลังอนุมัติแล้ว กดเพื่อสั่งพิมพ์ (อัปเดตสถานะอัตโนมัติ)

### 5.3 สถานะผลตรวจเลือด (`/test-status`)

ติดตามสถานะการดำเนินการของผลตรวจทุกราย:
- กรองตามสถานะ
- อัปเดตสถานะขั้นตอน

## 6. การพิมพ์ผลตรวจ

### 6.1 พิมพ์รายบุคคล (จากหน้า `/results`)

1. คลิกปุ่ม **"พิมพ์"** ข้างชื่อผู้ป่วย
2. ระบบจะแสดง **ตัวอย่างก่อนพิมพ์** (Preview Modal)
3. ตรวจสอบไฟล์ผลตรวจ
4. กดปุ่ม **"พิมพ์"** เพื่อยืนยัน
5. ระบบจะเปิดหน้าพิมพ์ **และ** อัปเดตสถานะผู้ป่วยเป็น **"รายงานผล"** โดยอัตโนมัติ

### 6.2 พิมพ์หลายคน (Batch Print)

1. **เลือกผู้ป่วย**: ติ๊กช่อง ☑️ ข้างชื่อแต่ละคน (หรือกด "เลือกทั้งหมด")
2. กดปุ่ม **"ดูตัวอย่าง & พิมพ์ (N)"** ที่ Panel ด้านขวา
3. ระบบจะแสดง **ตัวอย่างทุกคนที่เลือก** ในหน้าต่างเดียว
4. ตรวจสอบแล้วกดปุ่ม **"พิมพ์ (N รายการ)"**
5. แต่ละคนจะพิมพ์แยกหน้า (Page Break)
6. สถานะทุกคนจะอัปเดตเป็น **"รายงานผล"** อัตโนมัติ

### 6.3 พิมพ์จากหน้ารายละเอียด (`/results/[hn]`)

1. เข้าหน้าผลตรวจของผู้ป่วย
2. แพทย์/พยาบาลกดปุ่ม **"อนุญาตให้พิมพ์"** (ขั้นตอนอนุมัติ)
3. เมื่ออนุมัติแล้ว จะแสดงสถานะ **"พร้อมพิมพ์"** และปุ่ม **"🖨️ พิมพ์ผลตรวจ"**
4. กดพิมพ์ → ระบบเปิดหน้าพิมพ์ + อัปเดตสถานะ "รายงานผล"

> ⚠️ **สำคัญ**: การสั่งพิมพ์ = การยืนยันว่าผลตรวจเสร็จสมบูรณ์ สถานะจะเปลี่ยนเป็น "รายงานผล" ทันที

## 7. คิวงาน Lab (`/admin/lab/queue`)

> สำหรับ **Lab Staff** และ **Admin** เท่านั้น

### วิธีใช้งาน:

1. ดูรายชื่อผู้ป่วยที่รอผลตรวจ
2. เลือกผู้ป่วยที่ต้องการอัปโหลดผล
3. กดปุ่ม **"อัปโหลดผลตรวจ"**
4. เลือกไฟล์:
   - รูปภาพ (JPG, PNG, WebP)
   - PDF
   - Excel (.xlsx, .xls) — จะแปลงเป็นรูปอัตโนมัติ
5. เลือก **สรุปผล**:
   - 🟢 ปกติ (Normal)
   - 🟡 ผิดปกติ (Abnormal)
   - 🔴 วิกฤต (Critical)
6. กดปุ่ม **"อัปโหลด"**
7. ระบบจะส่งแจ้งเตือนไปยังเจ้าหน้าที่ที่รับผิดชอบ

### ตัวเลขแจ้งเตือน (Badge):

- ที่เมนู **"คิวงาน Lab"** ด้านซ้าย จะแสดง **ตัวเลขสีแดง** แสดงจำนวนผู้ป่วยที่รอผลตรวจ
- นับรวมสถานะ: เจาะเลือด + กำลังจัดส่ง + กำลังตรวจ
- อัปเดตอัตโนมัติทุก 30 วินาที

## 8. กล่องข้อความ (Inbox)

- **ข้อความใหม่**: แจ้งเตือนเมื่อมีผลตรวจพร้อม หรือข้อความจากเพื่อนร่วมงาน
- **อ่าน/ตอบ**: คลิกข้อความเพื่ออ่านและตอบกลับ
- **ป้ายแจ้งเตือน**: ตัวเลขสีแดงที่เมนูจะแสดงจำนวนข้อความที่ยังไม่ได้อ่าน
